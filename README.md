# Spot the Difference - Premium Web Game 🎮

실제 사진 이미지에서 무작위로 추출된 사물 형태를 기반으로 화사한 카툰화 또는 색상 변형(Hue Shift)을 거친 가상의 틀린 영역을 만들어 브라우저에서 틀린그림찾기를 즐길 수 있는 모던 프리미엄 웹 게임입니다.

---

## ✨ 주요 특징
1. **사물 외곽선(Contour) 기반 정밀 변형**:
   - 네모 박스 형태로 부자연스럽게 색칠되는 한계를 극복하고, 컴퓨터 비전(`Canny Edge` + `Morphology`) 연산을 통해 **실제 사물의 정확한 윤곽선 마스크**를 추출하여 해당 사물만 자연스럽게 변형 및 스무스 블렌딩합니다.
2. **2종의 고품질 변형 필터 무작위 적용**:
   - **카툰 일러스트 필터**: 얇고 선명한 드로잉 펜선과 쨍한 수채화 색감.
   - **사물 Hue Shift 필터**: 형태 및 광택, 명암은 그대로 유지하면서 원본 대비 뚜렷하게 대조(최소 60도 이상 회전)되는 쨍한 색상으로 변환.
3. **로비 & 테마 선택 대시보드**:
   - 이미지의 하위 폴더 구조(예: `stage1`, `stage2`, `앨범` 등)를 탐색하여 카테고리별 스테이지 선택 기능 제공.
4. **인터랙티브 포인터 동기화**:
   - 좌/우 이미지 뷰어의 마우스 조준경 위치가 100% 동일 비율로 연동되어 비교 플레이 지원.
5. **골드 네온 링 힌트 시스템**:
   - 테마당 3회의 힌트를 제공하며, 사용 시 1.5초간 해당 물체 위에 노란색 점선 네온 가이드를 비추고 페이드아웃 처리.
6. **윈도우 전용 무설치 구동 스크립트**:
   - 로컬 브라우저 CORS 보안 정책(로컬 JSON 읽기 실패)을 완벽히 우회하도록 데이터 브릿지 생성 및 배치 파일로 간편 구동 지원.

## 📂 프로젝트 파일 구조
```
FindDifference/
│
├── Images/
│   ├── Original/             # 원본 사진 이미지 폴더 (하위 서브폴더 구성 가능)
│   ├── Changed/              # 부분 변형 처리가 적용된 결과 폴더 (동일 서브폴더 복제)
│   ├── diff_coords.json      # 변형된 정답 영역 좌표 데이터 (x, y, w, h)
│   └── diff_coords.js        # 로컬 브라우저 CORS 우회용 JS 좌표 브릿지 데이터
│
├── index.html                # 게임 UI 레이아웃 및 HTML5 구조
├── style.css                 # 다크모드 글래스모피즘 및 애니메이션 CSS
├── app.js                    # 게임 루프, 스케일 비율 계산 및 클릭 판정 JS
├── generate_difference.py    # OpenCV 기반의 자동 윤곽선 검출 및 변형 파이썬 스크립트
├── play_game.bat             # 원클릭 웹 서버 실행 및 게임 시작 배치 파일
├── regenerate_images.bat     # 기존 변경 이미지를 비우고 새로 빌드하는 배치 파일
└── .gitignore                # 업로드 시 무거운 이미지 파일(Original/Changed) 제외 설정
```

## 🚀 시작하기

### 1. 게임 실행하기 (play_game.bat)
[play_game.bat](play_game.bat) 파일을 더블클릭하여 실행합니다.
- 자동으로 로컬 포트 `8000` 웹 서버가 열리며 웹 브라우저(`http://localhost:8000`)에 게임 화면이 실행됩니다.
- 게임을 완전히 끄려면 실행된 검은색 커맨드 프롬프트 창을 닫아주시면 됩니다.

### 2. 새 이미지 추가 및 전체 재생성 (regenerate_images.bat)
- `Images/Original/` 내부에 나만의 사진(하위 폴더를 추가하셔도 좋습니다)을 자유롭게 배치합니다.
- [regenerate_images.bat](regenerate_images.bat) 파일을 더블클릭하여 실행합니다.
- 기존의 Changed 이미지들과 좌표 데이터가 깔끔하게 리셋된 후, 추가된 원본 사진들에 맞춰 물체 Contour 감지 및 대조 색조 튜닝이 적용된 새로운 변경 이미지가 자동 재생성됩니다.
