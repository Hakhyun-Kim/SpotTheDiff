# Spot the Difference - Premium Web Game 🎮

실제 사진 이미지에서 무작위로 추출된 사물 형태를 기반으로, 컴퓨터 비전 기술을 통해 화사한 만화 스티커 또는 부분 만화화 변형 영역을 생성하여 즐기는 프리미엄 틀린그림찾기 웹 게임입니다.

---

## ✨ 주요 특징

1. **사물 외곽선(Contour) 기반 정밀 변형**:
   - 네모 박스 형태로 부자연스럽게 변형되는 한계를 극복하고, 컴퓨터 비전(`Canny Edge` + `Morphology`) 연산을 통해 **실제 사물의 정확한 윤곽선 마스크**를 추출하여 자연스러운 합성 및 스무스 블렌딩을 적용합니다.
2. **로비 & 테마 선택 대시보드**:
   - 업로드된 이미지의 하위 폴더 구조(예: `stage1`, `stage2`, `앨범` 등)를 탐색하여 카테고리별 테마 카드를 로비에 자동으로 구성합니다.
3. **웹 기반 간편 이미지 업로더**:
   - 게임 최초 실행 시 이미지가 없으면, 화면에 웹 브라우저 업로더가 표시되어 이미지 폴더 또는 개별 이미지 파일을 드래그 앤 드롭해 바로 추가할 수 있습니다.
   - 로비의 **[이미지 추가]** 버튼을 통해 언제든지 새로운 이미지를 손쉽게 추가할 수 있습니다.
4. **실시간 변경 이미지 자동 생성**:
   - 업로드된 이미지들은 백엔드 Flask 서버에서 즉시 OpenCV를 통해 틀린 그림 처리 및 정답 좌표를 생성하며, 새로 고침 없이 로비에 즉각 스테이지가 구성됩니다.
5. **인터랙티브 포인터 동기화**:
   - 좌/우 이미지 뷰어의 마우스 조준경 위치가 100% 동일 비율로 연동되어 정밀한 비교 플레이를 지원합니다.
6. **골드 네온 링 힌트 & 피드백 시스템**:
   - 테마별 3회의 힌트를 제공하며, 사용 시 1.5초간 노란색 점선 네온 가이드를 비춥니다. 클릭 시에는 confetti 효과가 발산되어 플레이 만족감을 극대화합니다.

## 📂 프로젝트 파일 구조

```
SpotTheDiff/
│
├── Images/
│   ├── Original/             # 원본 사진 이미지 폴더 (사용자 업로드 시 서브폴더 구성 가능)
│   ├── Changed/              # 부분 만화화 스티커가 적용된 변경 이미지 폴더 (동일 서브폴더 복제)
│   ├── Stickers/             # 합성에 사용되는 카툰 스티커 이미지 소스 (duck, heart, star 등)
│   ├── diff_coords.json      # 변형된 정답 영역 좌표 데이터 (x, y, w, h)
│   └── diff_coords.js        # 로컬 브라우저 CORS 우회용 JS 좌표 브릿지 데이터
│
├── index.html                # 게임 UI 레이아웃 및 HTML5 구조
├── style.css                 # 다크모드 글래스모피즘 및 애니메이션 CSS
├── app.js                    # 게임 루프, 화면 탐색, 드래그 앤 드롭 업로드 및 클릭 판정 JS
├── generate_difference.py    # OpenCV 기반의 자동 윤곽선 검출 및 변형 파이썬 스크립트
├── server.py                 # Flask 기반의 웹 서버 및 이미지 업로드/재생성 API
├── play_game.bat             # 원클릭 웹 서버 실행 및 게임 시작 배치 파일
├── regenerate_images.bat     # 기존 변경 이미지를 비우고 새로 빌드하는 배치 파일
└── .gitignore                # 업로드 시 무거운 이미지 파일(Original/Changed) 제외 설정
```

## 🚀 시작하기

### 1. 게임 실행하기 (play_game.bat)
- [play_game.bat](file:///d:/SpotTheDiff/play_game.bat) 파일을 더블클릭하여 실행합니다.
- 자동으로 Flask 로컬 서버(`http://localhost:6001`)가 실행되며, 웹 브라우저 창이 열립니다.
- 게임을 완전히 종료하려면 커맨드 프롬프트 창을 닫아주시면 됩니다.

### 2. 최초 실행 시 이미지 추가 (자동 셋업)
- 만약 원본 사진 파일이 하나도 없는 채로 처음 시작하면, 웹 브라우저에 **이미지 업로드 화면**이 즉시 나타납니다.
- `폴더 선택` 또는 `파일 선택` 버튼을 누르거나 이미지 파일을 영역 안으로 드래그 앤 드롭한 뒤, **[업로드 및 생성 시작]**을 클릭합니다.
- 업로드 완료 즉시 틀린그림찾기 이미지 및 정답 좌표가 자동 생성되어 로비 화면으로 전환됩니다.

### 3. 게임 중 추가 이미지 등록
- 로비 화면 상단의 **[이미지 추가]** 버튼을 통해 언제든지 새로운 이미지를 테마에 실시간으로 등록하고 틀린그림찾기를 확장할 수 있습니다.

## 🛠️ 요구 사항

- **Python 3.x**
- **필수 라이브러리**: Flask, OpenCV 및 Pillow, numpy
  - 배치 스크립트 실행 시 미설치 라이브러리가 존재하면 자동으로 설치를 시도하거나, 아래 명령어로 직접 수동 설치할 수 있습니다:
    ```bash
    pip install flask opencv-python pillow numpy
    ```
